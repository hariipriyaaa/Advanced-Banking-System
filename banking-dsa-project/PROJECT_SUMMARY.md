## 🎯 Key Features Implemented

### 1️⃣ Data Structures
- **Trie** - Suspicious pattern matching (4 fraud patterns)
- **Max Heap** - Top-K suspicious accounts ranking
- **LRU Cache** - O(1) account lookups
- **Graph** - Transfer network (adjacency list)
- **Deque** - Sliding window implementation
- **HashMap** - Customer database

### 2️⃣ Algorithms
- **Sliding Window** - O(n) fraud detection
- **Dijkstra's Shortest Path** - O((V+E) log V) optimal routing
- **Trie Search** - O(m) pattern matching
- **Heap Operations** - O(log k) updates

### 3️⃣ Full-Stack Features
- ✅ Customer account management
- ✅ Deposits, withdrawals, transfers
- ✅ Real-time fraud detection
- ✅ Transaction history
- ✅ Year-wise analytics
- ✅ Suspicious account ranking
- ✅ Interactive dashboard

```bash

./setup.sh

# 3. Start backend (Terminal 1)
cd backend && npm start

# 4. Start frontend (Terminal 2)
cd frontend && npm start
```

**Open:** http://localhost:3000

## 🐛 Troubleshooting

### Port Already in Use
```bash
# Kill process on port 5000
kill -9 $(lsof -ti:5000)

# Kill process on port 3000
kill -9 $(lsof -ti:3000)
```

### Compilation Error
```bash
# Make sure you have C++17 support
g++ --version

# Should show GCC 7.0 or higher
```

### npm install fails
```bash
# Clear cache
npm cache clean --force

# Try again
npm install
```
